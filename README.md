# Khwaish Goel — Portfolio

A cinematic, interactive personal portfolio built with React and Vite. Features a physics-based skills globe, AI-powered typing effects, and a full red-black glassmorphism design system.

🔗 **Live Site:** [khwaish-portfolio.vercel.app](https://khwaish-portfolio.vercel.app)

---

## Features

- **Intro Screen** — Dramatic enter screen with ambient glow and cinematic video intro that plays once per session
- **Home** — Full-width autoplay video with slow-motion effect and infinite typewriter bio
- **Skills Globe** — Physics-based interactive ellipse globe with 45+ skill pills bouncing, colliding and floating in real time using Canvas API
- **Projects** — Glassmorphism project cards with live demo and GitHub links
- **Navigation** — Slide-in vertical navbar with red hover effects, icons and logo
- **Footer** — Animated contact section with social links

---

## Tech Stack

**Frontend**
- React 18
- Vite
- React Router DOM
- HTML5 Canvas API (custom physics engine)

**Design**
- Custom CSS (no UI library)
- Cormorant Garamond + DM Mono fonts
- Glassmorphism + red/black theme
- Fully responsive — mobile to desktop

**Deployment**
- Vercel

## Run Locally
```bash
# Clone the repo
git clone https://github.com/Khwaish06/khwaish-portfolio.git

# Navigate into the project
cd khwaish-portfolio

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
